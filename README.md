## 2. Brend identifikatsiyasi

### 2.1 Logotip
Fayl: `assets/brand/logo-source.jpg` (Telegram kanal rasmidan olingan, past piksellashgan — ishlab chiqarish uchun mijozdan yuqori sifatli original so'rash tavsiya etiladi).

- **"SUPER-TOUR.UZ"** — qalin, katta harflar, to'q ko'k rang
- **"TRAVEL AGENCY"** — yupqa (light), keng harf oralig'i (letter-spacing), kulrang-ko'k, kichikroq o'lchamda, tagida ajratuvchi chiziqlar bilan

## 4. Texnik eslatmalar (keyingi bosqich uchun)

- Stack: **React (Vite) + Node.js/Express** — bonoos-landing loyihasidagi struktura asos qilib olinadi (`client/` + `server/`, production'da Express `client/dist`ni serve qiladi).
- Til: kamida rus tili; o'zbek tili qo'shish rejalashtirilsa — i18n uchun oddiy JSON-based tarjima tuzilmasi tavsiya etiladi.
- Rasm/media: hozircha real fotolar yo'q (Instagram bloklandi, Telegram faqat past sifatli thumbnail berdi) — MVP uchun mavzuga mos stok-fotolar bilan boshlanadi, keyin mijozdan asl fotolar/logotip fayli so'rab almashtiriladi.
- "Ommabop yo'nalishlar" va narxlar bo'limi kelajakda kanal postlaridan avtomatik yangilanadigan qilib qurilishi mumkin (masalan, admin panel yoki oddiy JSON konfiguratsiya orqali) — hozircha statik kontent sifatida boshlanadi.
